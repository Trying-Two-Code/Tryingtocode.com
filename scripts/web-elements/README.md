# Dependancies

All components should need the custum-elements.css folder to be included. Any css for web elements should be in there.

# Conventions

Use ttc- as the beginning of every custom element to make it very clear that it is a custom element made specifically for tryingtocode.com

# Important notes

Not all custom elements will need imports, but if they do, you must use (type="module" + defer) when using the script in html. Example: <script src="example" type="module" defer></script>
